# 功能清单

本项目是一个以「恋人 X」人设陪伴用户「轩」的 AI 陪伴聊天应用。
前端：React 18 + Vite 5；后端：Node.js 原生 HTTP（`last/server.local.js`，端口 8888）；数据存本地 SQLite（`.local-storage/data.db`，`lib/storage.js` 键值表，底层 better-sqlite3）；AI 走 OpenAI 兼容接口（火山方舟）。

以下为**已实现**的全部功能。

---

## 一、聊天对话

1. 发送文本消息并保存到后端；无会话时自动创建首个会话。（`App.jsx`）
2. AI 流式回复（SSE），边接收边打字机展示。（`App.jsx` + `lib/api.js` + 后端 `/api/chat/stream`）
3. 保留非流式对话接口作为回退。（后端 `/api/chat`）
4. 多轮上下文组装，仅取最近 20 条历史控制 token。（`App.jsx`）
5. 发送失败兜底：try/catch/finally 复位「输入中」状态、回填输入内容方便重发。（`App.jsx`）
6. 「正在输入」三点动画气泡。（`TypingIndicator.jsx` / `ChatArea.jsx`）
7. 消息已读/未读状态显示（用户消息按是否有 AI 回复、AI 消息按页面可见性判断）。（`ChatArea.jsx` + `Message.jsx`）
8. 智能滚动，浏览历史时显示「新消息 / 回到底部」浮动按钮。（`ChatArea.jsx`）
9. 工具调用过程可视化时间线（输入 / 输出 / 成功失败）。（`Message.jsx`）
10. 多行自适应输入框、回车发送、模型选择下拉。（`InputArea.jsx`）
11. 会话动态标题与在线状态点。（`ChatHeader.jsx`）

## 二、AI 主动消息与通知

12. 后端每 1 小时扫描所有会话，按条件（超 3 小时冷静期、当天未达 5 条上限、15% 概率、北京时间 7:00–23:00 活跃时段）生成「突然想起你」的主动消息。（`lib/memory/proactive.js`）
13. 前端每分钟轮询当前会话，发现 `proactive` 主动消息即刷新界面。（`App.jsx`）
14. 主动消息去重，避免历史消息重复提示。（`App.jsx`）
15. 首次进入请求浏览器通知授权；收到新主动消息弹出浏览器 Notification。（`App.jsx`）

## 三、AI 提供商配置

16. AI 接入健康状态检查与重新检查。（`AIConfigPanel.jsx` + 后端 `/api/ai-status`）
17. 预设 9 类服务商模板（火山 / OpenAI / Claude / 智谱 / 豆包 / 通义 / 零一 / DeepSeek / 自定义）自动填充端点。（`AIConfigPanel.jsx`）
18. 添加 / 启用停用 / 删除 / 测试连接 AI 提供商；API Key 明文存储、对外返回打码。（`AIConfigPanel.jsx` + 后端 `/api/ai-providers`）
19. 主 AI（对话）与辅助 AI（记忆压缩等后台任务，用低成本模型、不占主 AI Token）分离。（`lib/ai-provider.js`）
20. 已启用模型进入聊天页模型下拉供切换。（`App.jsx` + `InputArea.jsx`）

## 四、工具调用（工作间）

21. 工具列表按分类筛选、启用/停用、显示已启用数量。（`ToolConfigPanel.jsx` + 后端 `/api/tools`）
22. 新增自定义工具：普通工具 / AI 技能 / MCP 服务 / 本地手机 App（可配接口地址、启动命令、目标 App 与动作）。（`ToolConfigPanel.jsx`）
23. 聊天时把已启用工具以 OpenAI function calling 格式传给 AI，支持两轮工具调用（请求工具 → 执行 → 再总结）。（后端 `server.local.js` + `lib/tools.js`）

内置工具：
- **网页搜索** `web_search`：博查 Web Search API，返回前 5 条带摘要结果。
- **打开网页** `open_webpage`：抓取指定 URL 正文，防 SSRF、15 秒超时、正文截断约 4000 字。
- **计算器** `calculator`：数学表达式求值（字符白名单校验）。
- **天气查询** `weather_query`：wttr.in 返回温度、天气、湿度、风速。
- **翻译** `translator`：MyMemory API，支持中英日韩法德西俄等互译。
- **系统时间** `system_time`：返回北京时间日期/时间/星期。
- **代码执行** `execute_code`：JavaScript 受限沙箱；Python 经 `/api/execute-code` 真实执行本机子进程。

## 五、记忆系统

24. 记忆增 / 查 / 改 / 删、搜索（内容或来源）。（`MemoryPanel.jsx` + 后端 `/api/memories`）
25. 置顶（不衰减）、标记已解决（降权）、归档 / 恢复。（`MemoryPanel.jsx`）
26. 重要度、情感效价（valence）、唤醒度（arousal）滑块设置。（`MemoryPanel.jsx`）
27. 标签增删（右键 / 长按）、按状态 / 来源 / 标签筛选、跨会话开关。（`MemoryPanel.jsx`）
28. 记忆详情弹窗、命中次数、复制会话 ID。（`MemoryPanel.jsx`）
29. 新增记忆时自动情感分析打标、关键词自动打标签、语义去重（相似度 ≥ 0.6 判重）。（`lib/memory/compress.js` + `core.js`）
30. 增强版语义检索浮现：规则分（重要性/激活次数/情感强度/时间衰减）50% + 语义相似度 50%，置顶强制保留，含跨会话记忆合并。（`lib/memory/surface.js`）
31. 智能遗忘曲线：超 7 天未激活的非置顶记忆按 5% 速率降权。（`lib/memory/core.js`）
32. 从对话抽取「喜好」「日程」两类精确事实，存为独立高优先级记忆。（`lib/memory/compress.js`）
33. 记忆压缩：达阈值后把最早的历史对话以恋人 X 视角压缩成一条摘要记忆（用辅助 AI）；自动与手动两种触发。（后端 `server.local.js` + `/api/chats/{id}` compress）
34. 跨会话身份关联（如 Web 与 QQ 会话映射），支持跨端记忆共享。（`lib/memory/core.js`）

## 六、日记 / 周记 / 月记

35. 按月浏览日记，分组展示日记 / 周记 / 月记，心情图标由情感值推断。（`DiaryPanel.jsx`）
36. 手写日记：标题、日期、心情、标签、正文，支持编辑 / 删除。（`DiaryPanel.jsx`）
37. 按指定日期触发「AI 整理」自动生成日记。（`DiaryPanel.jsx` + 后端 `/api/diary/compile`）
38. 多级记忆金字塔：日常记忆 → 日记（每天 0 点自动整理）→ 周记（每周浓缩，置顶）→ 月记（每月 ≥4 篇周记时浓缩，最高层级）。（`lib/memory/diary.js`）
39. 启动时回溯最多 30 天，为有记忆但缺日记的日期补生成。（`lib/memory/diary.js`）
40. 日记状态查询（总篇数、今日是否已生成、最近日记日期）。（后端 `/api/diary/status`）

## 七、共读伴侣（一起读）

41. 上传并解析 TXT / 文字版 PDF（浏览器端 pdfjs 提取文字），保存为读书笔记。（`ReadingPartner.jsx`）
42. 读书笔记新建 / 编辑 / 删除，类型分高亮 / 疑问 / 感悟，标签增删。（`ReadingPartner.jsx` + 后端 `/api/notes`）
43. 就某条笔记与 AI 讨论，长文只发「开头 + 相关片段」节选防 token 浪费。（`ReadingPartner.jsx`）
44. 笔记及讨论同步到记忆系统（单条 / 一键批量同步）。（`ReadingPartner.jsx`）
45. 「去和她讨论」把笔记内容带入主聊天页发送（直接调 `sendText`，不依赖 DOM 模拟按键）。（`ReadingPartner.jsx` + `App.jsx`）
46. localStorage 离线缓存与首次上云迁移。（`ReadingPartner.jsx` + 后端 `/api/notes/bulk`）

## 八、设置

47. 对话设置：聊天对象名、系统提示词（人设）、Temperature、Top P、最大 Token、记忆压缩阈值、压缩后保留轮数。（`SettingsPanel.jsx` + 后端 `/api/settings`）
48. 应用设置：语言切换、通知开关、自动保存（存 localStorage）。（`AppSettingsPanel.jsx`）
49. 底层人设规则来自 `base-rules.md`，对话与主动消息时拼接进 system prompt。（后端）

## 九、导航与主页

50. 首页「欢迎回家」：最近会话预览卡片、四大功能模块入口、实用工具栏。（`HomePage.jsx`）
51. 会话侧边栏：会话列表切换、新建、删除、重命名（编辑框 / 右键 / 长按菜单）、压缩记忆入口。（`Sidebar.jsx`）
52. 首页与聊天页切换、各功能面板开关。（`App.jsx`）

## 十、应用自检与错误监控

53. 基础健康检查：后端服务、数据库、AI API、工具配置、代码执行环境。（`AppCheckPanel.jsx`）
54. 实时错误监控列表，按严重 / 警告 / 运行时 / API 筛选，含原因与解决建议。（`AppCheckPanel.jsx` + `lib/errorMonitor.js`）
55. fetch 拦截自动记录 API 错误，全局 error / 未捕获 Promise 捕获。（`lib/errorMonitor.js`）
56. 心跳检测后端在线 / 断线 / 恢复，展示运行时长、最后心跳。（`lib/errorMonitor.js` + 后端 `/api/heartbeat`、`/api/logs`）

## 十一、PWA / 移动端

57. 移动端响应式适配：弹窗改底部抽屉、圆角与尺寸自适应（窗口 <768px 检测）。（`MemoryPanel.jsx`、`DiaryPanel.jsx`、`ReadingPartner.jsx`）
58. 触摸交互：长按弹出菜单、`navigator.vibrate` 震动反馈。（多组件）
59. Capacitor Android 打包（`first/android/`）。

## 十二、存储与基础设施

60. 本地 SQLite 键值存储（chats / memories / ai-providers / settings / tools / reading-notes 等，每类一行、value 存 JSON），底层 better-sqlite3，WAL 模式 + 事务防写坏，读写容错。（`lib/storage.js`）
61. 数据导出 / 导入备份：一键导出全部数据为 JSON 备份文件下载、选择备份文件导入恢复（导入前覆盖确认、校验键名防路径穿越）。（后端 `/api/export`、`/api/import` + `AppCheckPanel.jsx` 数据备份区）
62. JSON→SQLite 一次性迁移脚本，把旧的 `.local-storage/*.json` 导入数据库（`npm run migrate`）。（`scripts/migrate-json-to-sqlite.js`）
63. 服务器内存日志（最多 100 条）供 `/api/logs` 读取。（`lib/storage.js`）
64. CORS 配置（从环境变量读允许源，默认 `*`）。（`lib/storage.js`）
65. 设置默认值集中管理（temperature、max_tokens、top_p、memory_threshold、keep_recent_messages、memory_decay_rate 等）。（`lib/storage.js`）
66. 敏感配置（`ARK_API_KEY`、`BOCHA_API_KEY`、`HELPER_AI_PROVIDER_ID` 等）均通过环境变量注入，不硬编码。
